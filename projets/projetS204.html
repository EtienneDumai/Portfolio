    Description :

    Le but de ce projet était d'utiliser python pour créer des graphiques à partir de fichiers csv. Ensuite dans une deuxième partie nous devons faire la même chose mais intégrer une partie base de données pour faire une application en console, qui demande si on veut des graphiques ou bien une valeur précise de la représentation du graphique.


    • Utilisation de fichiers csv
    • Application avec base de données


    Première partie :

    La première partie consistait à créer des graphiques à partir de fichiers csv. Pour cela, j'ai utilisé la librairie pandas pour lire les fichiers csv et stocker les données dans des dataframes. Ensuite j'ai utilisé la librairie matplotlib pour créer les graphiques.
